## API Access

The MarketDataAPI provides REST endpoints for:

- **Instruments**: Financial instruments (equities, bonds, derivatives)
- **Legal Entities**: Company information via LEI codes
- **MIC Codes**: Market identification codes and trading venues
- **Transparency**: MiFID II transparency calculations
- **File Management**: ESMA FIRDS/FITRS data processing
